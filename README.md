# react-native-next-monorepo

![Repository Banner](https://i.imgur.com/wPdflX8.png[/img])

This is a basic monorepo for React Native and Next.js.

## Getting Started

Fork or clone this repo and then use

```
yarn all
```

To run the app in iOS, Android & web.

OR

```
yarn ios
yarn android
yarn web
```
